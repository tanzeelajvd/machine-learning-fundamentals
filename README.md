# Machine Learning Fundamentals

This repository contains **core undergraduate-level study material for Machine Learning**, combining **theoretical foundations** with **hands-on implementations**.

The course follows the official **CAP781: Machine Learning** syllabus and is designed to build a strong understanding of classical and modern machine learning techniques through structured lectures and practical notebooks.

## üìò Course Coverage

The course covers the following major areas of Machine Learning:

### Core Topics

* Introduction to Machine Learning
* Supervised Learning

  * Linear, Polynomial, and Nonlinear Regression
  * Logistic Regression
  * k-Nearest Neighbors (k-NN)
  * Support Vector Machines (SVM)
  * Decision Trees and Random Forests
  * Ensemble Methods (Bagging, Boosting)
* Model Evaluation and Hyperparameter Tuning

### Unsupervised Learning

* Clustering (k-Means, Hierarchical)
* Dimensionality Reduction (PCA)
* Feature learning and anomaly detection

### Image Processing and Computer Vision

* Digital image processing techniques
* Feature extraction and segmentation
* Applications in computer vision

### Neural Networks and Deep Learning

* Perceptrons and Multi-Layer Perceptrons
* Activation functions
* Convolutional Neural Networks (CNNs)
* Recurrent Neural Networks (RNNs)
* Transfer learning and pre-trained models

### Advanced Topics

* Reinforcement Learning
* Q-Learning and Deep Q Networks
* Model optimization and compression
* Emerging trends in Machine Learning
